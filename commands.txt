[gpon]
show gpon onu uncfg :: Показати незареєстровані ONU на PON
show pon power attenuation gpon-onu_1/1/1:1 :: Показати сигнал ONU абонента
show pon power onu-rx gpon-olt_1/1/1 :: Показати сигнали всіх ONU на PON
show gpon onu state gpon-olt_1/1/1 :: Показати кількість активних та зареєстрованих ONU на порту
show running-config interface gpon-onu_1/1/1:1 :: Показати конфіг ONU
show gpon onu detail-info gpon-onu_1/1/1:1 :: Показати логи падіння та підняття ONU
show clock :: Показати час на OLT
show gpon remote-onu interface eth gpon-onu_1/1/1:1 :: Перевірка зєднання між ONU та роутером (швидкість)
show mac gpon onu gpon-onu_1/1/1:1 :: Показати Mac адресу роутера
show ip dhcp snooping dynamic port pon gpon-onu_1/1/1:1 vport 1 :: Показати Mac адресу, IP та VLAN роутера
show ip dhcp snooping port gpon-onu_1/1/1:1 :: Показати Mac адресу, IP та VLAN роутера
show pon power onu-rx gpon-olt_1/1/1 :: Показати сигнали всіх ONU на порту
show gpon onu by sn abcd.efgh.1234 :: Показати порт на якому зареєстрована ONU
show mac vlan 99 :: Показати всі MAC адреси всіх ONU які зареєстровано на VLAN 99
show vlan summary :: Показати всі номери VLAN на OLT
show card slotno 1 :: Показати інформацію про OLT (Скільки часу він працює)

conf t :: Зайти в налаштування OLT (Список команд)
> ip dhcp snooping clear pon abcd.efgh.1234 vlan 111 gpon-onu_1/1/1:1 vport 1 :: Очистка сесії
> ip dhcp snooping clear abcd.efgh.1234 vlan 111 :: Очистка сесії
> interface gpon-onu_1/1/1:1 :: Зайти в інтерфейс ONU
> no onu 111 :: Видалення ONU з PON
> onu 111 type GPON-1GE sn PTNHUYLO :: Реєстрація ONU за серійним номером на PON
> mac add permanent abcd.efgh.1234 gpon gpon-onu_1/1/1:1 vport 1 vlan 99 :: Привязати MAC роутера перманентно.
> mac delete abcd.efgh.1234 :: Видалити перманентно привязану MAC адресу
pon-onu-mng gpon-onu_1/1/1:1 :: Перехід в режим конфігурації ONU (Список команд)
> interface eth eth_0/1 state lock/unlock :: Очистка сесії без відєднування ethernet кабелю.
> reboot :: Перезавантаження ONU віддалено.
